# Template Editor Constitution

Updated: 2026-04-23
Status: Governing template-authoring interaction law.

## 1. Editor Authority

### 1.1 Clause IDs

- `TMP-ED-001`: The template editor shall be full-screen.
- `TMP-ED-002`: The template editor shall behave as a governed WYSIWYG authoring surface.
- `TMP-ED-003`: The template editor shall not function as a side panel that hides structural control.

### 1.2 Editing constraints

- `TMP-ED-010`: Drag and drop shall be allowed only within governed section constraints.
- `TMP-ED-011`: The editor shall not permit broken layout rules.
- `TMP-ED-012`: The editor shall validate template compliance live.

### 1.3 Output parity

- `TMP-ED-020`: The editor view shall align with PDF output law.
- `TMP-ED-021`: Editor previews shall not diverge from governed template rules.
- `TMP-ED-022`: Editor controls shall be explicit, measurable, and reversible.
