# Inventory Control Point Seeds

Updated: 2026-04-23
Status: Seed document for inventory control-point generation.

## 1. Seed Scope

- `INV-CP-001`: These seeds shall be used to generate future inventory control points.

## 2. Seed Categories

- `INV-CP-010`: Stock mismatch.
- `INV-CP-011`: Missing movement logs.
- `INV-CP-012`: Negative stock violation.
- `INV-CP-013`: Missing cost data.
- `INV-CP-014`: COGS mismatch.
- `INV-CP-015`: Inventory-accounting mismatch.

## 3. Seed Expectations

- `INV-CP-020`: Each control point shall include control id, source clause id, module, title, purpose, expected behavior, failure condition, severity, measurable fields, evidence requirement, likely root-cause zones, and retest requirement.
- `INV-CP-021`: Each control point shall be traceable to a governing clause.
- `INV-CP-022`: Each control point shall be machine-auditable.
