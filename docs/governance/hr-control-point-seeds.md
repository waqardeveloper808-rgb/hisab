# HR Control-Point Seeds

## 1. Scope
1.1 These seeds define control definitions for future HR control points.

## 2. Seed List
- `HR-CP-001`: Missing employee data
- `HR-CP-002`: Payroll mismatch
- `HR-CP-003`: Attendance inconsistency
- `HR-CP-004`: Accounting mismatch

## 3. Required Seed Fields
3.1 Each future control point shall include control id, source clause id, module, title, purpose, expected behavior, failure condition, severity, measurable fields, evidence requirement, likely root-cause zones, and retest requirement.

## 4. Clause IDs
4.1 Stable clauses in this document use the prefix `HR-CP`.

## 5. Enforceable Clauses
- `HR-CP-001`: HR control points shall be clause-linked.
- `HR-CP-002`: HR control points shall be measurable.
- `HR-CP-003`: HR control points shall require evidence.
- `HR-CP-004`: HR control points shall define retest requirements.
