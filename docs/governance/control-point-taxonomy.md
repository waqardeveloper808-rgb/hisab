# Control Point Taxonomy

## Categories
1. Constitutional controls
1.1 Enforce supremacy, inheritance, traceability, and lifecycle law.

2. Workflow controls
2.1 Enforce route, shell, posting, reconciliation, and lifecycle behavior.

3. Financial controls
3.1 Enforce accounting, VAT, journal, ledger, payroll, and reporting truth.

4. Compliance controls
4.1 Enforce ZATCA, EU readiness, and governed document finalization.

5. Rendering controls
5.1 Enforce preview, PDF, template, and invoice output equivalence.

6. Integrity controls
6.1 Enforce stock, balance, and data consistency across modules.

7. Traceability controls
7.1 Enforce source clause, source record, audit, and evidence lineage.

8. Reporting controls
8.1 Enforce drill-down, report-to-ledger reconciliation, and read-only truth.

9. UX controls
9.1 Enforce workspace shell, responsiveness, interaction, and preview flow.

10. HR and payroll controls
10.1 Enforce employee, attendance, payroll, and payroll accounting linkage.

11. Governance controls
11.1 Enforce precedence, runtime binding, module acceptance, and dashboard visibility.
